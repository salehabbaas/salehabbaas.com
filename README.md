# salehabbaas.com

Production-ready Next.js + Firebase system with:

- Public personal site (Apple-like visual direction)
- Admin panel with Firebase Auth + custom claim guard
- Creator OS (admin management + public SEO pages)
- Job application tracker (admin-only) + XLSX export

## Quick start

1. `cp .env.example .env.local`
2. `npm install`
3. `cd functions && npm install && cd ..`
4. `npm run dev`

Detailed deployment/setup: see [`RUNBOOK.md`](./RUNBOOK.md)
